## Proyecto Sprint 7

# Análisis de información sobre vehículos usados.

Esta aplicación servirá para presentar un Análisis Exploratorio de Datos (EDA) que busca explicar como se relacionan el kilometraje y el precio en el mercado automotriz.

Se presentan visualizaciones especificas:
- Un histograma del odometro, el cual muestra como se distribuyen los kilometros de los autos en el conjunto de datos
- Un grarfico de dispersión que permite observar la relación entre el kilometraje y el precio de los vehiculos. 
- Tambien encontramos una tabla que nos muestra el promedio del precio y el kilometraje de los autos y su variación año tras año. 

Se agregó un botón para la creación de los diferentes graficos en la aplicación.
